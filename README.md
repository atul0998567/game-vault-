# GameVault

## Project Overview
GameVault is a web application that helps users discover classic and modern video games. Users will be able to search for games, filter results by platform, sort by rating or release date, save favorite games, and compare selected titles side by side.

## Purpose
The purpose of this project is to create a user-friendly game discovery platform where users can quickly explore video game information in one place. The app is designed for gamers who want to search for titles, browse details, and organize games they may want to play later.

## Public API Used
This project uses the RAWG Video Games Database API:

https://rawg.io/apidocs

The RAWG API provides access to a large collection of video game data, including:
- Game titles
- Release dates
- Ratings
- Platforms
- Screenshots
- Genres
- Descriptions
- System requirements

## Planned Features
- Search for video games by title
- Debounced search input for better user experience
- Filter games by platform such as PC, PlayStation, and Xbox
- Sort games by rating, popularity, or release date
- Display games in a responsive grid layout with box art and ratings
- Save selected games to a Favorites or Wishlist section using browser storage
- Compare two games side by side to view key details
- View detailed information for each game
